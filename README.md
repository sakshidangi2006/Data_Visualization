# 📊 Data Visualization — IPL Capstone Project

A collection of Jupyter Notebooks exploring data visualization techniques using IPL (Indian Premier League) cricket data. This project covers everything from basic Matplotlib plots to interactive Plotly dashboards.

---

## 📁 Repository Structure

```
Data_Visualization/
│
├── IPL.csv                                  # Raw IPL dataset
├── IPL_Capstone_Project.ipynb               # Main capstone notebook
├── Distrubution_And_Categorical_Plots.ipynb # Distribution & categorical analysis
├── Matplotlib.ipynb                         # Matplotlib fundamentals
├── MatrixPlot.ipynb                         # Correlation matrix visualizations
├── RegressionPlot.ipynb                     # Regression analysis & plots
└── plotly.ipynb                             # Interactive Plotly charts
```

---

## 📓 Notebooks Overview

| Notebook | Description |
|---|---|
| `IPL_Capstone_Project.ipynb` | End-to-end analysis and visualization of IPL data |
| `Distrubution_And_Categorical_Plots.ipynb` | Histograms, KDE plots, bar charts, and count plots |
| `Matplotlib.ipynb` | Core Matplotlib concepts — line, scatter, bar, and pie charts |
| `MatrixPlot.ipynb` | Heatmaps and pairplots to explore feature correlations |
| `RegressionPlot.ipynb` | Linear regression visualizations using Seaborn/Matplotlib |
| `plotly.ipynb` | Interactive and animated plots using Plotly Express |

---

## 📦 Dataset

**`IPL.csv`** — Contains match-level IPL data including teams, venues, toss results, match outcomes, player of the match, and more.

---

## 🛠️ Tech Stack

- **Python 3.x**
- **Pandas** — Data manipulation
- **Matplotlib** — Static visualizations
- **Seaborn** — Statistical plots
- **Plotly** — Interactive charts
- **Jupyter Notebook** — Development environment

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/sakshidangi2006/Data_Visualization.git
   cd Data_Visualization
   ```

2. **Install dependencies**
   ```bash
   pip install pandas matplotlib seaborn plotly notebook
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. Open any `.ipynb` file to explore the visualizations.
